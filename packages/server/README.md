# server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.20. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

## Prisma

bunx prisma init

### Migrations

bunx prisma migrate dev

### Generate Migrations

bunx prisma generate
